# ITP/SEW Vocabulary Trainer

## Description

This project is a simple Java program for school.  
It is a vocabulary trainer for ITP/SEW technical terms.

The program asks the user technical words and the user has to enter the correct answer.  
It works like a small quiz in the console.

---

## Purpose

The goal of this project is to:

- Practice Java programming  
- Learn and repeat ITP/SEW technical terms  
- Work with user input in Java  

---

## How to Run

1. Download or clone the project  
2. Open it in a Java IDE  
3. Run the main class  
4. Answer the questions in the console  

---

## Technologies

- Java  
- Object-Oriented Programming  
- Scanner for user input
